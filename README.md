# yapv
A viewer for [push_swap](https://github.com/thberrid/push_swap), a sorting stacks program.

[Preview](http://pushswap.dafox.co)

# usage

    npm install
    node server.js

Apache  configuration

	ProxyRequests Off
	ProxyPass / http://localhost:8080/
	ProxyPassReverse / http://localhost:8080/
